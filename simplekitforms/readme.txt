Create simple forms and manage responses directly from WordPress.

== Description ==

Simple Kit Forms is a streamlined plugin for creating simple forms and managing responses directly from your WordPress installation. It manages its own database tables for forms and entries — all data stays on your server.

== Installation ==

1. Upload the `simplekitforms` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to SK Forms to create your first form.
