Generate QR codes for WordPress posts and pages with download in PNG 2048x2048px and access tracking.

== Description ==

Simple Kit QRCode generates QR codes for your WordPress posts and pages, making it easy to share links via print materials, posters, business cards, or any offline medium. Each QR code includes a tracking URL that lets you monitor how many times it has been accessed.

= Features =

* QR code generation for posts and pages
* High-resolution PNG download (2048x2048px)
* Access tracking for each QR code
* Post/page search and filtering
* Statistics page with access counts
* Backup and restore support
* Standards-compliant QR codes via QRServer API with built-in fallback

== Installation ==

1. Upload the `simplekitqrcode` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to SK QRCode to generate your first QR code.

== Frequently Asked Questions ==

= How do I generate a QR code? =

Go to SK QRCode, select a post or page from the list, and click "Generate QR Code". A tracking URL is encoded in the QR code so you can monitor access counts.

= How do I download a QR code? =

After generating a QR code, click "Download QR Code PNG (2048x2048px)" to save the high-resolution image to your computer.

= How do I track accesses? =

Go to SK QRCode → Statistics to see how many times each QR code has been scanned. Each access is automatically counted when a visitor follows the tracking URL.
